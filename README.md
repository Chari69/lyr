# Lyr

> [!NOTE]
> This project was partially generated by an AI model: Github Copilot

Programa de Python para mostrar eltexto de un archivo `.txt` en una interfaz con fondo verde. Las líneas se muestran una a una y avanzan al presionar Enter.

Se usó para evitar tener que editar y sincronizar la letra de una canción.

## Requisitos
- Python 3.10+

## Arquitectura

El proyecto ahora está modularizado:

- `lyr.py`: punto de entrada.
- `app/application.py`: bootstrap de la aplicación.
- `app/model/lyrics_model.py`: lógica de dominio y estado de líricas.
- `app/controller/lyrics_controller.py`: coordinación entre modelo y vista.
- `app/view/lyrics_view.py`: interfaz Tkinter, animaciones y controles visuales.

## Uso

Desde PowerShell o CMD en Windows:

```powershell
# Opción A: Pasando el archivo por argumento
python lyr.py sample_lyrics.txt

# Opción B: Sin argumento (se abrirá un diálogo para elegir el .txt)
python lyr.py
```

Atajos:
- Enter: siguiente línea
- Ctrl+O: abrir archivo

## Personalización en la interfaz

- Selector de fuente en pantalla.
- Importación de fuentes `.ttf/.otf` desde el programa (se copian en `app/assets/fonts/`).
- Control de tamaño de texto (14 a 96).
- Botón `Aplicar` para usar los cambios.
- Selector de animación de salida y entrada (independientes).
- Tipos de animación: `none`, `fade`, `zoom`, `tiktok`.
- Duración configurable para salida y entrada (0 a 2000 ms).
- Botón `Aplicar animación` para confirmar cambios.
- Color de texto personalizable.
- Borde configurable (grosor + color).
- Color de fondo personalizable (default: verde original).
- Botón `Aplicar estilo` para actualizar colores y borde.
- Menú `Ayuda > Acerca de` con información del programa.
